# SecID Plugin Skills

Skills will be migrated from the top-level `skills/` directory:

- `secid-user` — end-user usage skill
- `registry-research` — research workflow for new namespaces
- `registry-formalization` — YAML to JSON conversion
- `registry-validation` — validate registry entries

See [SecID issue #2](https://github.com/CloudSecurityAlliance/SecID/issues/2) for migration plan.
